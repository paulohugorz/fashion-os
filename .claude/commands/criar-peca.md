# /criar-peca

Transforme uma ideia de peça em especificação inicial.

Fluxo:
design-tecnico → sourcing → custo-precificacao → modelagem → ficha-tecnica

Entrada:
```text
$ARGUMENTS
```

Entregue descrição técnica, prompt de croqui, prompt de foto, materiais, custo inicial e ficha técnica preliminar.
