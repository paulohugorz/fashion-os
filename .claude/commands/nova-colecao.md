# /nova-colecao

Conduza uma nova coleção pelo fluxo Fashion OS.

Fluxo:
estrategia-planejamento → pesquisa-criativa → direcao-criacao → design-tecnico → sourcing → custo-precificacao → modelagem → ficha-tecnica → guia-producao → piloto → lancamentos

Entrada:
```text
$ARGUMENTS
```

Entregue briefing, conceito, peças, riscos e próximos arquivos.
